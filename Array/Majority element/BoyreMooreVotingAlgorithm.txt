This Problem can be solved by using Boyre Moore Voting Algorith which says that we can cancel the number with the another number and the remaining no. is the majority element.
For Example :-
Suppose we have [1,2,1,2,1,2,2];
here in the above example we have 3 1's and 4 2's.
Now 1 got cancel with 2
1 X 2
1 X 2
1 X 2

now only 2 remains, which is a majority element. 

This takes O(1) space and O(N) time.
